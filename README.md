# achievement-challenge
Achievement Challenge

Technologies Used:
- Client:
  - HTML5
  - CSS3
  - JavaScript
  - jQuery 2.1.4
- Server:
  - Node.js v4.1.2
  - ECMAScript 6
  - MongoDB 3.0.6
- Both:

  - WebSocket
